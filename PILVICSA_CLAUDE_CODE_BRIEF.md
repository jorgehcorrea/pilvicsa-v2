## 1. Project overview

Rebuild the Pilvicsa website (`pilvicsa.com.ec`) from scratch using Tailwind CSS v4 + Flowbite + vanilla JS. The current site runs on a patched Canvas Bootstrap template with a fragile product database. The rebuild must be clean, fast, maintainable, and bold in visual style — inspired by agrotech startup aesthetics.

The site is a **static HTML site** — no server-side rendering, no CMS, no database. All product data lives in `data/products.json` which is generated from a Google Sheets CSV by a build script. The site deploys as plain files to cPanel.

**Primary goals:**
- Product catalogue with 221+ varieties searchable via Fuse.js fuzzy search
- Full-screen video hero on homepage
- Social media post embeds (Instagram, Facebook, YouTube) per product
- Google Sheets → CSV → JSON pipeline so products can be updated without touching code
- Image optimization pipeline (raw photos → uniform WebP)
- Bold agrotech visual identity — dark sections, large typography, strong green palette

## 5. data/products.json schema

This file is the single source of truth for all product data. Generated by `scripts/fetch_sheet.py`. Never edit manually.

```json
[
  {
    "id": "hortalizas_001",
    "nombre": "Tomate Lupitas F1",
    "latin": "Solanum lycopersicum",
    "categoria": "Tomates Variedades francas e injertadas",
    "pagina": "hortalizas",
    "pageLabel": "Hortalizas",
    "color": "Rojo",
    "forma": "",
    "sistema": "Invernadero",
    "densidad": "",
    "resistencias": "TM2, TSWV",
    "ciclo": "75 dias",
    "tamano": "180-220 gr",
    "imagen": "hortalizas_001.webp",
    "youtube_id": "",
    "fb_video_id": "",
    "ig_post_id": "",
    "estado": "confirmed",
    "nota": ""
  }
]
```

**Rules:**
- Empty string fields `""` mean the field is absent — do NOT render blank rows in UI
- `pagina` maps to which product page the item belongs to
- `pageLabel` is the human-readable category name shown in UI
- `imagen` is filename only — path is always `images/productos/{imagen}`
- If `imagen` is empty, show a green placeholder with a leaf icon

## 7. Scripts

### scripts/fetch_sheet.py
```python
"""
Fetches Pilvicsa product data from Google Sheets published CSV.
Usage: python scripts/fetch_sheet.py
Output: data/products.json
"""
import requests, csv, json, io, os

SHEET_CSV_URL = "REPLACE_WITH_PUBLISHED_CSV_URL"
OUTPUT_PATH = os.path.join(os.path.dirname(__file__), '..', 'data', 'products.json')

COLUMNS = ['id','nombre','latin','categoria','pagina','pageLabel','color','forma',
           'sistema','densidad','resistencias','ciclo','tamano','imagen',
           'youtube_id','fb_video_id','ig_post_id','estado','nota']

def fetch():
    r = requests.get(SHEET_CSV_URL, timeout=30)
    r.raise_for_status()
    reader = csv.DictReader(io.StringIO(r.text))
    products = []
    for row in reader:
        p = {col: row.get(col, '').strip() for col in COLUMNS}
        if p['id'] and p['nombre']:  # skip empty rows
            products.append(p)
    with open(OUTPUT_PATH, 'w', encoding='utf-8') as f:
        json.dump(products, f, ensure_ascii=False, indent=2)
    print(f"Written {len(products)} products to {OUTPUT_PATH}")

if __name__ == '__main__':
    fetch()
```

### scripts/optimize_images.js
```javascript
/**
 * Converts all images in images/raw/ to WebP 800x600 in images/productos/
 * Usage: node scripts/optimize_images.js
 * Requires: npm install sharp
 */
const sharp = require('sharp');
const fs = require('fs');
const path = require('path');

const RAW_DIR = path.join(__dirname, '..', 'images', 'raw');
const OUT_DIR = path.join(__dirname, '..', 'images', 'productos');
const SUPPORTED = ['.jpg','.jpeg','.png','.webp','.heic','.tiff'];

if (!fs.existsSync(OUT_DIR)) fs.mkdirSync(OUT_DIR, { recursive: true });

const files = fs.readdirSync(RAW_DIR).filter(f => 
  SUPPORTED.includes(path.extname(f).toLowerCase())
);

(async () => {
  for (const file of files) {
    const name = path.parse(file).name;
    const outPath = path.join(OUT_DIR, name + '.webp');
    try {
      await sharp(path.join(RAW_DIR, file))
        .resize(800, 600, { fit: 'cover', position: 'centre' })
        .webp({ quality: 82 })
        .toFile(outPath);
      console.log(`✓ ${file} → ${name}.webp`);
    } catch(e) {
      console.error(`✗ ${file}: ${e.message}`);
    }
  }
  console.log(`Done. ${files.length} images processed.`);
})();
```

### Fuse.js configuration
```javascript
const fuse = new Fuse(PILV_DB, {
  keys: [
    { name: 'nombre', weight: 3 },
    { name: 'latin', weight: 2 },
    { name: 'categoria', weight: 1.5 },
    { name: 'color', weight: 1 },
    { name: 'resistencias', weight: 1 },
    { name: 'sistema', weight: 1 },
    { name: 'nota', weight: 0.5 }
```
